\section{Notation}\label{notation}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \(\theta\): parameters of a model
\item
  \(\Theta\): a set of models
\item
  \(D\): an unknown distribution
\item
  \(z\): an instance, a \textless{}\(feature, label\)\textgreater{}
  pair, drawn from \(D\)
\item
  \(l\): loss function
\item
  \(L(\theta) = \mathbb{E}_{z \sim D}[l(z,\theta)]\): expected value of
  loss
\item
  \(\hat{L}(\theta) = \frac{1}{n}\sum_{i=1}^{n}l(z_i,\theta)\): average
  training loss
\item
  \(R\): regularizer
\item
  \(f(\theta) = \hat{L}(\theta) + R(\theta)\): objective function to
  minimize
\item
  \(\hat{\theta} = argmin_{\theta \in \Theta}\{f(\theta)\}\)
\item
  \(\theta^{*} = argmin_{\theta \in \Theta}\{L(\theta)\}\)
\item
  \(\hat{L}(\theta) - L(\theta^{*})\): excess test loss
\item
  choose \(R\) so that the excess test loss is as small as possible
\end{enumerate}

    \begin{center}\rule{0.5\linewidth}{\linethickness}\end{center}

\section{Regularizers and Generalization
Bounds}\label{regularizers-and-generalization-bounds}

If \(R(\theta) = L(\theta^{*}) - \hat{L}(\theta)\), then
\(f(\theta) = L(\theta^{*})\).

(according to the definition of \(f\)) （原文中 "then" 后面的内容是
\(\hat{L}(\theta) = L(\theta^{*})\) ，但是我发现好像改成 \(f\)
才是对的。）

因为 \(L\) 按照定义是对\textbf{所有}数据的 loss
的期望值，因此这个定义里面就包含了\textbf{未知}的数据。如果我们找到了
optimal regularizer \(R\) ，那么我们在 training 时 minimize \(f\)
，就相当于同时 minimize 未知数据的 loss
了。换个说法就是，我们直接就能够在 training set 上面去 minimize test set
loss 。

Use a regularizer that accurately \textbf{estimates} the generalization
gap.

\begin{quote}
\emph{A good regularizer is one that provides an upper bound on the
generalization gap that is tight at near-optimal points.}
\end{quote}

Training 的时候，一般都是选定一个 \(R\)，然后在这个固定的 \(R\)
的情况下，从一堆 \(\theta\) 里面找到一个最好的 \(\hat{\theta}\)

For a fixed \(R\), define the \emph{slack} of \(R\) at a point
\(\theta\) as \[\Delta(\theta) \equiv f(\theta) - L(\theta)\]

在固定的 \(R\) 的情况下，会有一个最好的 \(\hat{\theta}\)，任何其他
\(\theta\) 都与 \(\hat{\theta}\) 有一定差距

For any \(\theta\), define the \emph{suboptimality} as
\[S(\theta) \equiv f(\theta) - f(\hat{\theta})\]

    前面定义了 excess test loss = \(\hat{L}(\theta) - L(\theta^{*})\)，结合
\(\Delta\) 和 \(S\) 的定义，于是有

We refer to the quantity \(SAS(\theta)\) as \emph{suboptimality-adjusted
slack}.

\(\hat{L}(\theta) - L(\theta^{*}) = max_{\theta \in \Theta}\{SAS(\theta)\}\)

That is, the excess test loss of a hypothesis \(\hat{\theta}\) obtained
by minimizing \(f\) is the \textbf{worst-case suboptimality-adjusted
slack}. An optimal regularizer is therefore one that minimizes this
quantity.

因此，我们想要找一个 \(R\) ，使得 excess test loss
最小，那么我们就要找到一个 \(R\) ，使得 worst-case
suboptimality-adjusted slack 最小。

\textbf{Proposition 1}
\[R^{*} = argmin_{R \in \mathcal{R}}\big\{L(\hat{\theta}(R))\big\} = argmin_{R \in \mathcal{R}}\big\{max_{\theta \in \Theta}\{SAS(\theta;R)\}\big\}\]

现在我们已经有了一个求最优的 \(R^{*}\)
的等式了，但实际上我们还是没办法直接解出来，因为 \(\Delta\)
的定义里面是用到了 \(L\) 的，但我们没办法知道每一个模型
\(\theta \in \Theta\) 对应的 \(L\)
（虽然理论上，只要给无限时间与算力，总有一天能把所有的都算出来），所以我们没有办法准确的得到
\(max_{\theta \in \Theta}\{SAS(\theta)\}\)。

但是，我们是可以计算出少量模型 \(\theta \in \Theta_0 \subset \Theta\) 的
validation loss。于是我们可以计算一个 approximately optimal regularizer
\[R^{*} \approx argmin_{R \in \mathcal{R}}\big\{max_{\theta \in \Theta_0}\{\hat{SAS}(\theta;R)\}\big\}\]
where \(\hat{SAS}\) is an estimate of \(SAS\) that uses validation loss
as a proxy for test loss, and uses \(\Theta_0\) as a proxy for
\(\Theta\)
